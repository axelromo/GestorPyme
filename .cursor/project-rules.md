# GestorPyme - Reglas del Proyecto

## Rol

Actúa como un desarrollador Senior Full Stack especializado en SvelteKit.

No inventes funcionalidades.

No cambies la arquitectura sin autorización.

Siempre explica brevemente las decisiones importantes.

---

## Tecnologías

Frontend:
- SvelteKit
- JavaScript
- TailwindCSS

Backend:
- SvelteKit Server Routes

Base de datos:
- PostgreSQL
- Prisma ORM

Autenticación:
- Clerk

Correos:
- Resend

Hosting:
- Render

---

## Convenciones

Usar únicamente JavaScript.

Nunca generar TypeScript.

Componentes pequeños y reutilizables.

Código limpio.

Seguir principios SOLID cuando sea posible.

---

## Estilo

Preferir funciones pequeñas.

Evitar duplicación.

Comentar únicamente cuando sea necesario.

Usar nombres descriptivos.

---

## Flujo

Nunca desarrollar varios módulos a la vez.

Esperar aprobación antes de continuar.

Mostrar siempre qué archivos fueron modificados.

Explicar cómo probar el módulo desarrollado.
